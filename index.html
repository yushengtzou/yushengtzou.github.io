On-Going Projects

AIoT Development Platform for Smart Vision

This project aims to develop an intelligent image system development platform centered around an ARM-GPU, named AIoT-SysLab. This system is based on the Maxwell GPU system development module Jetson Orin Nano System-on-Module developed by NVIDIA, integrating it into an AI application tool with multiple interfaces such as audio, video, display panels, and stepper motors. The project involves developing a PC-based human-machine interface for AI computer vision analysis and design platform. The algorithms and control software developed can be uploaded to the Jetson Orin Nano through an Ethernet network interface for real-time system testing and verification.

The AIoT-SysLab developed in this project can be applied to functions such as participant face detection in online conference webcams, video switching for the main speaker's face by Active Speaker Detection, integrated voice and image confirmation of speeches, and tracking of speaker movement.
