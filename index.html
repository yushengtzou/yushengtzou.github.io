3D Face Model for Surgical Simulation and Visualization

This project aims at developing facial surgery simulation software named SimFace using 3D face model to simulate and visualize the post-surgery effect of face PDO thread-lifting micro-surgery. The main function of this software is using the system predefined "Facial Sculptor" tool to manipulate the expected shape of the 3D face model through the slider. Method used in the project may ranging from Finite Element Method(FEM) to Linear Elasticity method such as Regularized Kelvinlets. The developed software will be used at aesthetic clinic used by the surgeon acting as a communication tool with the customer pursuing for a better plastic surgery planning experiences.
